    GREEN: Create declarative_ai/execution/task_graph.py with compute_batches(tasks: list[TaskNode], completed_ids: set[str] | None = None) -> list[list[TaskNode]].

    Implementation (Kahn's algorithm):
    1. Build task_map (id -> TaskNode) and in_degree dict
    2. For each task's dependencies: if dep_id in completed_ids, skip (already satisfied); if dep_id not in task_map AND not in completed_ids, raise ValueError (unknown dep); otherwise increment in_degree
    3. Initialize queue with zero in-degree tasks
    4. BFS: drain queue into current batch, decrement dependents' in_degree, add newly-zero to next queue
    5. After loop: if completed count != len(tasks), raise ValueError (cycle detected)
    6. Return empty list for empty input

    Run tests — all must PASS.
